Buttons

다양한 크기, 상태 등을 지원하는 Qrame의 사용자 정의 버튼 스타일을 사용 하세요.

Example

Qrame Button을 사용하려면 class 이름을 <button> element에 추가 하세요.




                    
                    <button class="btn btn-default">Qrame Button</button>
                    <button class="btn btn-primary">Qrame Button</button>
                    <button class="btn btn-danger">Qrame Button</button>
                    <button class="btn btn-warning">Qrame Button</button>
                    <button class="btn btn-info">Qrame Button</button>

Note
  • "btn-default" class를 기본 버튼 스타일로 사용하며, "btn-primary" class를 검색, 저장, 신규 등에 강조 스타일로 사용 합니다.

Sizes

"btn-lg", "btn-sm", "btn-block" class로 크기를 조절 하십시오. 




                    
                    <button class="btn btn-default btn-lg">Qrame Button</button>
                    <button class="btn btn-primary btn-lg">Qrame Button</button>
                        



                    
                    <button class="btn btn-default btn-sm">Qrame Button</button>
                    <button class="btn btn-primary btn-sm">Qrame Button</button>
                        


                    
                    <button class="btn btn-default btn-block">Qrame Button</button>
                    <button class="btn btn-primary btn-block">Qrame Button</button>

Disabled

disabled속성을 <button> element에 추가하여 버튼이 비활성화 된 것처럼 보이게 하십시오. 




                    
                    <button class="btn btn-default" disabled>Qrame Button</button>
                    <button class="btn btn-primary" disabled>Qrame Button</button>

Image Button

"icon" class를 <button> element에 추가하여 이미지 버튼을 만들 수 있습니다.




                    
                    <button class="btn btn-warning btn-send-notice btn-notice-auto-confirm"></button>
                    <button class="btn btn-info btn-send-notice"></button>
                    <button class="btn btn-default btn-add-sum"></button>
                    <button class="btn btn-default btn-code-search"></button>
                        




                    <button class="btn btn-info btn-send-notice"> 확인</button>
                    <button class="btn btn-default btn-code-search"> 검색</button>

Button Group

Button을 Group으로 묶으세요. 




                    
                    <div class="btn-group">
                    <button class="btn btn-primary"></button>
                    <button class="btn btn-default">Qrame Button1</button>
                    <button class="btn btn-default">Qrame Button2</button>
                    </button>

    • "input-title" class : "intput-title" class를 사용해서 input영역의 Title 크기를 일정하게 맞춥니다.

Position

  • position-static : 정적 위치로 조정합니다.

  • position-relative : 상대적으로 배치 된 요소의 위쪽, 오른쪽, 아래쪽 및 왼쪽 속성을 설정하면 일반 위치에서 멀어 지도록 조정됩니다.

  • position-absolute : 절대 위치 요소에 위치 된 부모가 없으면 문서 본문을 사용하고 페이지 스크롤과 함께 이동합니다.

  • position-fixed : 페이지가 스크롤 되더라도 항상 동일한 위치에 유지됩니다.


                <div class="position-static"></div>
                <div class="position-relative"></div>
                <div class="position-absolute"></div>
                <div class="position-fixed"></div>
Overflow

  • overflow-auto : scroll하지만 필요할 때만 스크롤 막대를 추가합니다.

  • overflow-scroll : 오버플로가 잘리고 스크롤 막대가 추가됩니다.

  • overflow-hidden : 오버플로가 잘리고 나머지 내용은 보이지 않습니다.

  • overflow-visible : 오버플로가 잘리고 나머지 내용은 보이게 됩니다.


                <div class="overflow-auto"></div>
                <div class="overflow-scroll"></div>
                <div class="overflow-hidden"></div>
                <div class="overflow-visible"></div>
